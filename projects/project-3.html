Project 03

C-Based Elevator Control System

This project involves the design and implementation of an elevator management system using the C programming language. Developed as part of MREN 178, the system focuses on algorithmic efficiency and data structure application to handle multi-floor requests in real-time.

The control logic optimizes the elevator's pathing by managing a queue of passenger requests, ensuring minimal wait times and energy efficiency. It simulates a high-load environment where multiple inputs must be processed synchronously without system hang-ups.

C Programming Data Structures Queue Management Embedded Logic Algorithm Design
Request Optimization Utilizes custom data structures to sort floor calls and determine the most efficient travel direction.
Real-Time Simulation Built to handle concurrent inputs, simulating a real-world building environment with multiple floor requests.

Technical Implementation

01

Request Handling Logic

  • Implemented an arrival queue to store floor calls from passengers.
  • Developed a look-ahead algorithm to determine if the elevator should stop at a floor while already in motion.
02

Data Structure Design

  • Used Linked Lists/Queues for dynamic request management.
  • Optimized memory allocation to ensure the system remains stable over long periods of operation.
03

System Synchronization

  • Managed floor state transitions (Opening/Closing/Moving) using a Finite State Machine (FSM).
  • Handled "up" and "down" priority logic to mimic modern high-speed elevator behavior.

Core Skills

  • Advanced C Programming & Pointers
  • Dynamic Data Structure Implementation
  • Finite State Machine (FSM) Design
  • Algorithmic Optimization & Complexity Analysis

Development Challenges

  • Directional Conflict: Solving cases where passengers call for different directions simultaneously.
  • Concurrency: Ensuring that adding a new request mid-travel doesn't interrupt the current motor/simulation state.

Outcome

A fully functional simulation of an elevator controller that demonstrated a high level of proficiency in C programming and the practical application of data structures in mechatronic systems.

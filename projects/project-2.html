Project 02

Autonomous Self-Guided MiniBot

This project is a self-guided robot built to navigate an arena by following a high-contrast line. It uses a custom-built mechatronic claw for object manipulation and features a complete sensory feedback loop including obstacle detection and status indication.

The bot relies on dual color sensors positioned on the bottom for navigation, utilizing a "paddle" method where the sensors balance between black and white values to maintain pathing. An ultrasonic sensor acts as the primary eyes of the car, detecting obstacles in front and triggering the claw's action sequence.

Arduino Ultrasonic Sensing Dual Color Sensors Mechatronics State Feedback
Navigation Strategy Dual-sensor paddle logic follows a black line by sensing color transitions in real-time.
Visual Feedback Integrated LED array (Red, Yellow, Green) indicates the current state: moving, turning, or grabbing.

Technical Implementation

01

Sensory Wiring & Calibration

  • Integrated Ultrasonic sensor for front-end depth perception.
  • Wired status LEDs to specific GPIO pins to display real-time logic states.
02

Dual-Sensor Navigation

  • Developed logic to handle "half-black half-white" sensor inputs for smooth line tracking.
  • Mapped analog color sensor values to 0-3000 range for high-precision motor adjustment.
03

Automated Action States

  • Created a switch-case state machine for: Forward, Hard Turn, Slight Adjust, and Stop.
  • Synchronized motor stopping with the servo-claw "Lift and Curl" sequence.

Core Skills

  • Hardware Breadboarding & Wire Management
  • Embedded C++ Programming
  • Sensor Fusion (Ultrasonic + Color)
  • Mechatronic System Integration

Challenges & Solutions

  • Line Detection Latency: Optimized the map() functions to reduce computation time.
  • Obstacle Accuracy: Calibrated SENSOR_STOP boundaries to ensure the car stops exactly in front of objects.

Outcome

A successful build that demonstrated the integration of mechanical assembly, custom electronics, and real-time software control to achieve autonomous navigation and task completion.
